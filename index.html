高效Linux命令小结

  1. mv xxxx{,.bak} 等同于 mv xxxx xxxx.bak
  2. esc + . 能填充之前命令的最后一个字段
  3. ctrl + r然后输入xxx，搜索之前包含xxx的命令
  4. ctrl+z可以把当前程序丢后台，fg命令可以恢复。
    退出su，退出ssh，可以用ctrl+d(真有人不知道这个，手敲exit, logout)。
    sudo !!用sudo执行上条命令。
    如果在前台运行了一个程序，但是你需要退出终端，保持程序继续在后台运行，你可以这样: ctrl+z，把程序放在后台；bg让程序继续运行；disown -h %1（%n是你jobs命令返回的那个，没有其它后台程序的话，一般是%1）。
    cd - 返回上一个进入的目录
    !n ，重复执行第n条命令，如果该命令很长，就能节省大量输入命令的时间；
    10.!! ，重复执行上一条命令；
    sudo !! ，以管理员权限执行上一条命令；
    ps -eo pid,lstart|grep PID,查询进程的准确启动时间。
